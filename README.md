📊 Análise de Renúncias Fiscais no Brasil - 2024

Bem-vindo(a) ao meu primeiro repositório no GitHub! 🚀

Este projeto tem como objetivo analisar os dados abertos sobre as **Renúncias Fiscais** no Brasil referentes ao ano de 2024. A renúncia fiscal ocorre quando o governo abre mão de receber parte dos impostos para estimular a economia, desenvolver regiões ou apoiar programas sociais e entidades não governamentais.

## 🎯 Objetivo do Projeto
Transformar dados brutos do governo em informações claras e visuais, entendendo para onde estão sendo direcionados os incentivos fiscais do país através de análise de dados com Python e criação de dashboards em Power BI.

## 🛠️ Ferramentas Utilizadas
* **Python (Google Colab):** Utilizado para a limpeza, exploração e análise inicial dos dados, com destaque para o uso da função `groupby` para sumarização e agrupamentos.
* **Pandas:** Biblioteca Python para manipulação e estruturação dos arquivos CSV.
* **Power BI:** Ferramenta de Business Intelligence utilizada para a criação dos indicadores e do dashboard interativo final.

## 📂 Estrutura do Repositório
O repositório contém os seguintes arquivos:

* `analise_renuncias_fiscais_2024.ipynb`: Notebook (Google Colab) contendo a análise exploratória e os agrupamentos feitos em Python.
* `2024_RenunciasFiscais.csv`: Primeira base de dados extraída do Portal da Transparência, com os dados gerais das renúncias.
* `2024_RenunciasFiscaisPorBeneficiario.csv`: Segunda base de dados, detalhando as renúncias por beneficiário.
* `dashboard_renuncias_fiscais_2024.pbix`: Arquivo do Power BI com os indicadores visuais consolidados.

## 📈 Etapas da Análise
1. **Extração:** Download das bases de dados em formato CSV diretamente do Portal da Transparência do Governo Federal.
2. **Análise Exploratória (Python):** Importação dos dados no Google Colab e utilização da função `groupby` para entender os montantes de renúncia agrupados por categorias relevantes.
3. **Visualização de Dados (Power BI):** Construção de indicadores-chave (KPIs) e gráficos para facilitar a interpretação dos dados pelo público em geral.

* **Explicações Conceituais:** Portal da Transparência - [Entenda a Gestão Pública: Renúncias Fiscais](https://portaldatransparencia.gov.br/entenda-a-gestao-publica/renuncias-fiscais#:~:text=A%20ren%C3%BAncia%20fiscal%20ocorre%20quando%20o%20governo,setor%20privado%20ou%20por%20entidades%20n%C3%A3o%20governamentais)
